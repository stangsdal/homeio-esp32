# Homie for ESP8266 / ESP32

Welcome to the documentation for the maintained Homie fork used by the LSH
ecosystem.

This fork keeps the original Homie for ESP8266 API and runtime flow familiar,
while adding targeted fixes for current ESP32 and ESP8266 Arduino environments.
Homie `3.0.1` remains the default MQTT discovery convention. Homie `4.0.0` and
Homie `5.0` are available only as explicit build modes, so existing deployments
do not change behavior by accident.
